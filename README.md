# Chichat 
A chat website for hang out and talks with friends.

## Local development
First, run the development server:

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Application Stack
Backend:
	* Expressjs
	* Prisma

Frontend:
	* Nextjs
	* Tailwindcss
	* Framer-motion
	* Fornik
